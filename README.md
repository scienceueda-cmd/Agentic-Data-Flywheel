本システムは、Open-WebUIの拡張パックとして機能しつつ、ローカルPC上で**「半自動ファインチューニング」**と**「完全自律型AIエージェント」**の構築を可能にする学習循環（フライホイール）環境です。

日常のチャットやターミナル操作の履歴を自動で蓄積し、Llama 3などの軽量ローカルモデルに焼き付けることで、あなた専用の優秀なAIアシスタントを作り上げます。

## 💻 動作環境（必須システム要件）

ローカルPCで「LLMの推論」と「学習（ファインチューニング）」の両方を行うため、以下のスペックと環境が必要です。

- **OS**: Windows 10 または 11
- **Python**: **Python 3.12** （※システム内の実行スクリプトが `py -3.12` に固定されているため必須です）
- **GPU**: NVIDIA製グラフィックボード（**VRAM 8GB以上必須**。例：RTX 2070 SUPER, RTX 3060, RTX 4060 など）
- **ストレージ**: 10GB以上の空き容量（Hugging Faceから自動ダウンロードされるベースモデルの保存用）

## 📦 1. インストールと初回セットアップ

本システムは、既にあなたのPCに **Open-WebUI** がインストールされていることを前提としています。

1. プロジェクト直下にある **`Agent_Control_Panel.bat`** を開きます。
2. メニューから **`[5] Install Requirements`** を実行してください。（※現在のメニュー番号に読み替えてください）
3. **何がインストールされるの？**
   - AIモデルを動かすための `transformers`
   - 一般的なグラボ（VRAM 8GB）でも学習を可能にする軽量化ライブラリ `peft`, `unsloth`, `bitsandbytes`
   - エージェントの土台となる `langchain` などが一括でインストールされます。

## 🎮 2. クイックスタート：エージェントを使ってみよう

学習（ファインチューニング）を行う前に、まずは初期状態のエージェントを使ってみましょう。コントロールパネルには2つの使い道が用意されています。

### A. ターミナルで自律エージェントとして使う（メニュー `[3] Start CLI Agent`）
ターミナル上で直接AIに指示を出せます。このエージェントには初期状態で以下の**3つの超強力な標準ツール**が同梱されており、あなたのPCやネットの世界に干渉できます。
1. **`run_command`**: Windowsのコマンドプロンプトで任意のコマンドを実行（ファイル操作、Git操作など全般）。
2. **`view_file`**: 指定したテキストファイルやソースコードの中身を読み込む。
3. **`web_search`**: インターネット（DuckDuckGo）で最新情報やエラーの解決策を検索。

**【RAG機能（カンペ）も搭載】**
`.agents/skills/` フォルダにMarkdownファイル（コーディング規約やマニュアル等）を置いておくだけで、エージェントが起動時に自動で読み込みます。

### B. Open-WebUIと連携してチャットで使う（メニュー `[4] Start FastAPI Server for Open WebUI`）
Open-WebUIの画面から、このローカルAIと直接チャットができるようになります。
1. `[4]` を実行してAPIサーバーを起動したままにします。
2. Open WebUIの「設定」＞「接続」＞「OpenAI API」に以下を追加します。
   - **URL**: `http://localhost:8000/v1` / **API Key**: `sk-1234`（ダミーでOK）
3. モデル選択で `antigravity-agent` を選べばチャット可能です。

## 🔄 3. 本領発揮！データフライホイール（学習ループ）の回し方

日常的にエージェントを使っていると、自然とデータが溜まります。そのデータをAIの「脳みそ」に定着させる手順です。

### STEP 1: ログの自動監視をオンにする
コントロールパネルから **`[1] Setup Auto-Start for Monitor & Start Now`** を押します。
- これを押すと、Windows起動時に**裏側で自動的に監視スクリプトが立ち上がる**ようになります。
- 「Open-WebUIのチャット履歴」と「CLIエージェントの作業履歴」を監視し、**50KB（約2万5000文字）**溜まるごとに、自動で `ready_for_review/` フォルダの中に「パッケージファイル」として切り出し、Windows通知（トースト）でお知らせします。
- ※監視を完全に止めたい場合は、メニュー `[2] Disable Auto-Start & Stop Monitor` を押してください。

### STEP 2: マスターAIによるデータ精製（2つのルート）
抽出された生ログはそのままでは学習できません。「あなたがお使いのAI（マスターAI）」を使って、学習用データ（JSONL形式）に整形します。この時、**「エージェントとして学習させる（ReAct形式）」**か、**「普通のアシスタントとして学習させる（Chat形式）」**の2つのスタイルを選べます。

**ルートA：PCを操作できるAI（Claude CodeやAntigravity等）を持っている場合**
圧倒的に楽な方法です。あなたのマスターAIに以下の「魔法の1文」を投げるだけです。
> **「`D:\antigravity folder\agentic_finetuner\src\prompt_template.txt` を読み込んで実行して！」**
※AIが自動でデータを整形し、そのまま学習スクリプトの実行まで全自動で完了させてくれます！

**ルートB：ChatGPTやClaudeのウェブ版を使っている場合**
1. `ready_for_review/` に書き出された生ログを開いてコピーします。
2. ChatGPT等に、`src\prompt_template.txt` の内容（ルール）と、コピーした生ログを貼り付けて「このルールに従ってJSONLデータを作って」とお願いします。
3. 出力されたJSONLテキストをコピーし、プロジェクトの一番上のフォルダに `lora_dataset.jsonl` という名前で保存します。

### STEP 3: ファインチューニングの実行（ルートBの人のみ）
ルートBで手動保存した人は、コントロールパネルから **`[5] Start LoRA Fine-tuning`** を実行してください。
VRAM 8GBの限界ギリギリで学習が開始され、しばらく待つと新しい脳みそ（LoRAアダプター）が完成します！

## 🎮 4. コントロールパネル機能一覧

```text
[1] Setup Auto-Start for Monitor & Start Now
    -> ログ監視をWindowsスタートアップに登録し、今すぐ監視を開始。
[2] Disable Auto-Start & Stop Monitor
    -> スタートアップ登録を解除し、現在の監視も強制停止。
[3] Start CLI Agent
    -> ターミナル上で自律エージェントを起動（ツール使用可能）。
[4] Start FastAPI Server for Open WebUI
    -> Open-WebUI連携用のAPIサーバーを起動。
[5] Start LoRA Fine-tuning
    -> lora_dataset.jsonl を読み込んでファインチューニングを開始。
[6] Install Requirements
    -> 必須ライブラリの一括インストール。
[7] Exit
```

## ⚙️ 5. モデルの自動ダウンロードと設定変更（上級者向け）

このシステムは、初回起動時（エージェント起動や学習時）に、デフォルトのベースモデルとして `unsloth/llama-3-8b-Instruct-bnb-4bit` を Hugging Face から**自動でダウンロード**します（自分で手動ダウンロードする必要はありません！）。

もし「別のモデルをベースに学習させたい」場合や、「Cドライブの容量が少ないから別のドライブに保存したい」という場合は、プロジェクト直下にある **`config.json`** をメモ帳等で開いて書き換えてください。

```json
{
    "base_model_name": "unsloth/llama-3-8b-Instruct-bnb-4bit",
    "hf_cache_dir": "D:\\my_large_drive\\hf_cache"
}
```
- `base_model_name`: Hugging FaceのモデルID、またはローカルにダウンロード済みのモデルの絶対パス。
- `hf_cache_dir`: ネットから自動ダウンロードされたモデルデータが保存される場所。デフォルトはプロジェクト内の `./hf_cache` になっているので、USBメモリ等に入れて持ち運ぶことも可能です。
